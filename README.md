🎬 TMDB / Kinopoisk Clone

A movie catalog application built with React that uses the TMDB API to browse films, view details, and explore actors.

🚀 Tech Stack
React
TypeScript
Redux Toolkit (RTK Query + slices)
React Router
CSS Modules

⚙️ Features
Browse popular and trending movies
Movie details page with full information
Cast (actors) section
Integration with The Movie Database (TMDB) API
State management with Redux Toolkit
Client-side routing
Modular styling with CSS Modules
